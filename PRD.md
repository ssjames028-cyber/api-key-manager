**버전**: 1.0.0
**작성일**: 2026-03-30
**플랫폼**: macOS (Tauri v2)
**상태**: Draft

---

## 1. 개요 (Overview)

### 1.1 배경

개발자는 OpenAI, Anthropic, GitHub 등 수십 개의 서비스 API 키를 관리해야 한다.
현재는 텍스트 파일, 노션, 메모앱에 평문으로 저장하는 경우가 많아 유출 위험이 높다.

### 1.2 목표

로컬에서 완전히 동작하는 경량 API 키 금고(vault)를 제공한다.
키를 암호화하여 안전하게 보관하고, 필요할 때 한 번의 클릭으로 복사할 수 있게 한다.

### 1.3 비목표 (Out of Scope)

- 클라우드 동기화 / 다중 기기 지원
- 팀 공유 기능
- API 키 유효성 검증 (실제 API 호출)
- 브라우저 확장 연동
- Windows / Linux 지원 (v1)

## 3. 핵심 기능 요구사항 (Functional Requirements)

### 3.1 API 키 등록

| 항목 | 내용 |
|------|------|
| 진입점 | 우상단 "+ 새 키" 버튼 |
| 입력 필드 | 이름(Name), 값(Value) |
| 자동 기록 | 생성일시 (ISO 8601, 로컬 타임) |
| 유효성 검사 | 이름 필수, 값 필수, 이름 중복 불허 |
| 저장 방식 | 값은 AES-256-GCM으로 암호화 후 로컬 파일에 저장 |
| 성공 피드백 | 인라인 토스트 "키가 저장되었습니다" |

### 3.2 API 키 목록 조회

테이블 컬럼:

| 컬럼 | 설명 |
|------|------|
| 이름 (Name) | 등록한 서비스명 (예: OpenAI Prod) |
| 값 (Value) | 기본 마스킹 표시 (sk-p●●●●●●...7h...) |
| 생성일 (Created) | YYYY-MM-DD 형식 |
| 작업 (Actions) | 보기 / 복사 / 삭제 아이콘 |

- 기본 정렬: 생성일 내림차순 (최신 항목 상단)
- 빈 상태(empty state): "등록된 API 키가 없습니다. + 새 키를 추가해보세요." 안내 문구 표시

### 3.3 값 보기/숨기기 토글

- 눈 아이콘 클릭 → 평문 표시
- 눈-사선 아이콘 클릭 → 다시 마스킹
- 토글 상태는 앱 재시작 시 마스킹으로 초기화
- 한 행씩 독립적으로 토글 (전체 해제 버튼 없음)

### 3.4 클립보드 복사

- 복사 아이콘 클릭 → 복잠(decrypt) 후 클립보드에 평문 복사
- 30초 후 클립보드 자동 초기화 (보안)
- 복사 성공 시 아이콘이 체크 표시로 1.5초간 전환

### 3.5 API 키 삭제

- 휴지통 아이콘 클릭 → 확인 모달 표시
- 모달 내용: "'{이름}' 키를 삭제하시겠습니까? 이 작업은 되돌릴 수 없습니다."
- 버튼: [취소] [삭제]
- 삭제 후 목록에서 즉시 제거 + 토스트 "키가 삭제되었습니다"

### 3.6 이름 기반 검색

- 목록 상단 검색 입력창
- 실시간 필터링 (타이핑 즉시 반영, debounce 200ms)
- 검색어가 없으면 전체 목록 표시
- 대소문자 구분 없음 (case-insensitive)
- 결과 없음 시: "'{검색어}'에 해당하는 키가 없습니다." 표시

## 4. 보안 요구사항 (Security Requirements)

### 4.1 암호화 저장

| 항목 | 스펙 |
|------|------|
| 알고리즘 | AES-256-GCM |
| 키 파생 | PBKDF2 또는 기기 고유 식별자 기반 |
| 저장 경로 | `~/Library/Application Support/api-key-manager/vault.enc` |
| 평문 저장 금지 | 이름도 별도로 평문 인덱스에 저장 가능하나, 값(Value)은 절대 평문 저장 금지 |

### 4.2 화면 마스킹

- 기본 표시: `sk-p••••••••••••••••7h...` (앞 4자 + 마스킹 + 뒤 3자)
- 길이 정보 노출 최소화 (마스킹 길이 고정 12자)
- 스크린샷/화면공유 시에도 마스킹 상태가 기본

### 4.3 클립보드 보안

- 복사 후 30초 경과 시 클립보드 자동 초기화
- 타이머 진행 중 상태 표시 (복사 아이콘 옆 카운트다운)

### 4.4 앱 수준 보안

- Tauri allowlist: 파일 시스템 접근 최소 권한 (vault 경로만 허용)
- 네트워크 요청 전면 차단 (CSP + Tauri 네트워크 정책)
- 로그/콘솔에 키 값 출력 금지

## 8. 데이터 모델

```typescript
interface ApiKey {
  id: string           // UUID v4
  name: string         // 사용자 입력 이름 (평문)
  encryptedValue: string  // AES-256-GCM 암호화된 값
  createdAt: string    // ISO 8601
}

// vault.json 파일 구조 (복호화 후)
interface Vault {
  version: number      // 마이그레이션용
  keys: ApiKey[]
}
```

## 11. 수용 기준 (Acceptance Criteria)

- [ ] 키 등록 후 앱 재시작해도 목록에 유지됨
- [ ] 값 컬럼은 기본적으로 마스킹 처리됨
- [ ] 눈 아이콘 토글 시 해당 행만 평문/마스킹 전환
- [ ] 복사 후 30초 뒤 클립보드가 비워짐
- [ ] 삭제는 확인 모달 없이 단독으로 실행 불가
- [ ] 검색 시 매칭되지 않는 행은 즉시 숨겨짐
- [ ] vault 파일을 텍스트 에디터로 열면 키 값이 평문으로 보이지 않음
- [ ] 앱 번들 크기 10 MB 이하
- [ ] 네트워크 요청 0건 (Charles Proxy로 검증)
